# big CLI

Manage your [big](https://itsjustbig.com) Instagram DM inbox and automations from the terminal.

## Install

**Homebrew**
```sh
brew install D-C-Collectibles/big/big
```

**curl | bash**
```sh
curl -fsSL https://raw.githubusercontent.com/D-C-Collectibles/homebrew-big/main/install.sh | bash
```

Then:
```sh
big login          # paste a PAT from Settings -> API Tokens
big inbox list
```

Binaries are published to [Releases](https://github.com/D-C-Collectibles/homebrew-big/releases). Source lives in the private app repo.
